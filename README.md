# Software Engineering Practices

This repository contains lab assignments from the **Programming Practices** course at Wrocław University of Science and Technology (PWr). Each lab focuses on a core practice used in professional software development, implemented in **Python**.
 
---
 
## Labs
 
| # | Topic | Key Concepts |
|---|-------|--------------|
| [Lab 1](./lab1_tdd/) | **Test Driven Development** | Red–Green–Refactor cycle, unit testing |
| [Lab 2](./lab2_kata/) | **Pair Programming & Kata** | Driver/Navigator pattern, collaborative coding |
| [Lab 3](./lab3_code_review/) | **Code Review & Refactoring** | Peer review, clean code principles |
| [Lab 4](./lab4_design_patterns/) | **Design Patterns** | GoF patterns, OOP best practices |
| [Lab 5](./lab5_cicd/) | **CI/CD Pipeline** | GitHub Actions, automated testing & deployment |
| [Lab 6](./lab6_cloud/) | **Cloud & Automation** | Cloud environments, infrastructure scripting |
